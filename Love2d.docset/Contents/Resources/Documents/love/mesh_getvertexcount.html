Mesh:getVertexCount

Available since LÖVE 0.9.0
This function is not supported in earlier versions.

Gets the total number of vertices in the Mesh.

Function

Synopsis 

count = Mesh:getVertexCount( )

Arguments

None.

Returns

number count
The total number of vertices in the mesh.
