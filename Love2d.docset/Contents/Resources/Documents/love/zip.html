zip

The ZIP file format is a data compression and archive format. A ZIP file contains one or more files that have been compressed to reduce file size, or stored as-is. .love files are simply renamed .zip files.

Zip files can be read by the following programs:

  • 7-Zip
  • ALZip
  • Archive Utility
  • Ark
  • Bitser
  • bsdtar/libarchive
  • File Roller
  • Filzip
  • FreeArc
  • iArchiver
  • Info-ZIP
  • IZArc
  • KGB Archiver
  • PeaZip
  • PKZIP
  • PowerArchiver
  • StuffIt
  • The Unarchiver
  • TUGZip
  • WinAce
  • WinRAR
  • WinZip
  • XAD
  • Xarchiver
  • ZipGenius
