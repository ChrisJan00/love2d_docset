Lover

A lover is someone who uses LÖVE to create a game (/program/anything else), so, this is how it works:

Dev
Spawns LÖVE.
Lover
Develops games.
User
Plays games.

The name lover has been acknowledged as the best name for this in the Tuesday Convention.
