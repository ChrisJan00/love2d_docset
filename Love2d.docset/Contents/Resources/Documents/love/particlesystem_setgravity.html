ParticleSystem:setGravity

Removed in LÖVE 0.9.0
It has been replaced by ParticleSystem:setLinearAcceleration.

Sets the gravity affecting the particles (acceleration along the y-axis). Every particle created will have a gravity between min and max.

Function

Synopsis 

ParticleSystem:setGravity( min, max )

Arguments

number min
The minimum gravity.
number max (min)
The maximum gravity.

Returns

Nothing.
