love.image

Provides an interface to decode encoded image data.

Types

CompressedData Compressed image data designed to stay compressed in RAM and on the GPU. 0.9.0
ImageData Raw (decoded) image data.

Functions

love.image.newCompressedData Create a new CompressedData object from a compressed image file. 0.9.0
love.image.newEncodedImageData Encodes ImageData. 0.8.0
love.image.newImageData Creates a new ImageData object.
