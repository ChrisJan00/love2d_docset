Body:isAwake

Available since LÖVE 0.8.0
This method is not supported in earlier versions.

Returns the sleep status of the body.

Function

Synopsis 

status = Body:isAwake( )

Arguments

None.

Returns

boolean status
True if the body is awake or false if not.
