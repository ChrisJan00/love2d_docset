Löve2D docset

Offline documentation set of the Löve2D API in docset format for use with Zeal and similar applications. The content was imported from the Löve2D online wiki

Feed

Instructions: in Zeal, go to File->Options->Docsets->Add feed, paste the following URL:

dash-feed://http%3A%2F%2Fchrisjan00.github.io%2Flove2d_docset%2Flove2d.xml

Direct Download

Alternatively, download the tarball, unpack in your docset storage directory.

http://chrisjan00.github.io/love2d_docset/love2d.tgz
