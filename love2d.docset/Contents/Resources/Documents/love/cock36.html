cock.getMouseOffset

Gets horizontal and vertical mouse offset. You can access these values directly, see etc.mouse.* fields.

Function

Synopsis 

x, y = cock.getMouseOffset ( self )

Arguments

table self
An object to use.

Returns

number x
X-coordinate used as center.
number y
Y-coordinate used as center.
