enet.peer:state

Returns the state of the peer as a string.

Function

Synopsis 

peer:state()

Arguments

None

Returns

string state
The peer's current state. It can be any of the following:
  • "disconnected"
  • "connecting"
  • "acknowledging_connect"
  • "connection_pending"
  • "connection_succeeded"
  • "connected"
  • "disconnect_later"
  • "disconnecting"
  • "acknowledging_disconnect"
  • "zombie"
  • "unknown"
