Canvas:newImageData

Available since LÖVE 0.10.0
It has been renamed from Canvas:getImageData.

Generates ImageData from the contents of the Canvas.

Function

Synopsis 

data = Canvas:newImageData( )

Arguments

None.

Returns

ImageData data
The new ImageData made from the Canvas' contents.

Function

Synopsis 

data = Canvas:newImageData( x, y, width, height )

Arguments

number x
The x-axis of the top-left corner of the area within the Canvas to capture.
number y
The y-axis of the top-left corner of the area within the Canvas to capture.
number width
The width of the area within the Canvas to capture.
number height
The height of the area within the Canvas to capture.

Returns

ImageData data
The new ImageData made from the Canvas' contents.
