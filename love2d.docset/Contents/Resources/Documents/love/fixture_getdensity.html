Fixture:getDensity

Available since LÖVE 0.8.0
This method is not supported in earlier versions.

Returns the density of the fixture.

Function

Synopsis 

density = Fixture:getDensity( )

Arguments

None.

Returns

number density
The fixture density in kilograms per square meter.
