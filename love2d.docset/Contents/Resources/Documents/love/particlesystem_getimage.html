ParticleSystem:getImage

Available since LÖVE 0.9.0
This function is not supported in earlier versions.
Deprecated in LÖVE 0.9.1
This function will be removed in a future version. Use ParticleSystem:getTexture instead.

Gets the image used for the particles.

Function

Synopsis 

image = ParticleSystem:getImage( )

Arguments

Nothing.

Returns

Image image
An Image to use for the particles.
