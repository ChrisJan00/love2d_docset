Third Parties

LÖVE is currently using these libraries.

  • Lua - Scripting.
  • OpenGL - Hardware accelerated graphics.
  • DevIL - Image loading.
  • SDL - OS abstraction, mouse input, keyboard input.
  • LuaSocket - Network.
  • FreeType 2 - Font loading and rasterization.
  • PhysicsFS - Filesystem management + support for archive files.
  • OpenAL Soft - Audio playback.
  • mpg123 - MP3 support.
  • libogg/vorbis - OGG support.
  • libmodplug - Tracker formats support.
  • Box2D - Physics
